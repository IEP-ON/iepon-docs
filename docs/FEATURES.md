# IEPON 기능 상세 안내

## 🎯 현행수준 AI 분석

### 개요
IEPON의 핵심 기능으로, 학생의 현재 학습 수준을 AI가 자동으로 분석합니다.

### 분석 영역
- **국어**: 듣기·말하기, 읽기, 쓰기 영역
- **수학**: 수와 연산, 도형, 측정, 규칙성, 자료와 가능성 영역

### 분석 결과 제공 항목
1. 학습 특성 요약
2. 강점 영역
3. 보완 필요 영역
4. 권장 교수전략
5. 적정 성취기준 수준 제안

---

## 📝 교육계획 자동 생성

### 개요
현행수준 분석 결과를 바탕으로 월별·학기별 교육계획을 자동 생성합니다.

### 생성 내용
- 학기 교육 목표
- 월별 세부 목표
- 지도 방법 및 전략
- 평가 계획

### 출력 형식
나이스(NEIS) 개별화교육계획 입력 양식에 최적화된 형태로 제공됩니다.

---

## 📚 교과서 단원 연계

### 개요
IEPON만의 차별화된 기능으로, 2022 개정 교육과정 성취기준과 함께 **실제 교과서 단원 정보**를 제공합니다.

### 주요 기능
- 성취기준별 관련 교과서 단원 매핑
- 월별 교과서 단원 배정 기능
- 교육과정-교과서-IEP 삼각 연계

### 활용 방법
1. 학생의 현행수준에 맞는 성취기준 선택
2. 해당 성취기준과 연결된 교과서 단원 확인
3. 월별로 다룰 단원을 직접 배정
4. 실제 수업 진도와 일치하는 IEP 작성

## 📋 나이스 원클릭 복사

### 개요
생성된 모든 텍스트를 한 번의 클릭으로 클립보드에 복사합니다.

### 지원 항목
- 현행수준 분석 결과
- 학기별 교육 목표
- 월별 교육계획
- 교육평가 문구

### 복사 후 활용
나이스(NEIS) → 학생생활 → 개별화교육계획 → 해당 입력란에 붙여넣기

## 📱 모바일 관찰메모

### 개요
모바일 전용 인터페이스(`/m`)에서 학생 관찰 메모를 실시간으로 기록합니다.

### 주요 기능
- 실시간 관찰 메모 작성
- 날짜별 메모 정리
- 학생별 메모 분류
- 학기말 평가 시 참고 자료로 활용

### 활용 장면
- 수업 중 학생 행동 관찰
- 특별활동 참여 기록
- 상담 내용 메모
- 특이사항 기록

### PWA 설치
스마트폰 브라우저에서 "홈 화면에 추가"를 통해 앱처럼 설치하여 빠르게 접근할 수 있습니다.

---

## 📊 교육평가 보조

### 개요
학기말 교육평가 작성 시 AI가 평가 문구를 제안합니다.

### 분석 요소
- 학기 초 현행수준 데이터
- 학기 중 관찰메모
- 설정된 교육 목표

### 제공 내용
- 성취도 평가 문구
- 성장 과정 요약
- 다음 학기 권장 방향
