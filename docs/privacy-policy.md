# 개인정보처리방침

**IEPON (아이피온)** 서비스의 개인정보처리방침입니다.

**시행일**: 2024년 1월 1일  
**최종 수정일**: 2026년 1월 13일

---

## 1. 개인정보의 처리 목적

IEPON(이하 "서비스")은 개별화교육계획(IEP) 작성을 지원하는 AI 보조 웹 서비스로, **개인정보를 수집하거나 저장하지 않습니다**.

서비스는 다음의 목적을 위해 운영됩니다:
- 특수교사의 개별화교육계획 작성 지원
- AI 기반 현행수준 분석 및 교육계획 생성
- 교육평가 작성 보조
- 모바일 관찰메모 기록 지원

---

## 2. 개인정보의 수집 및 저장

### 2.1 개인정보 미수집 원칙

IEPON 서비스는 **회원가입 없이 사용 가능**하며, 다음과 같은 개인정보를 수집하거나 저장하지 않습니다:

- ❌ 사용자의 이름, 연락처, 이메일 등 개인 식별 정보
- ❌ 학생의 이름, 학번, 학교명 등 학생 개인정보
- ❌ 교육계획 내용, 평가 내용 등 교육 관련 데이터
- ❌ 관찰메모, 현행수준 분석 결과 등 서비스 이용 과정에서 입력된 모든 데이터

### 2.2 데이터 처리 방식

모든 데이터 처리는 **사용자의 브라우저(로컬 환경) 내에서만** 이루어집니다:

- ✅ 사용자가 입력한 모든 데이터는 브라우저의 로컬 스토리지에만 임시 저장됩니다
- ✅ 서버로 개인정보나 학생 정보가 전송되지 않습니다
- ✅ 브라우저를 닫거나 세션 종료 시 데이터가 자동 삭제됩니다
- ✅ AI 분석 시에도 개인 식별 정보는 제외되고 교육적 내용만 처리됩니다

### 2.3 AI 처리

AI 기능 사용 시:
- OpenAI API를 통해 교육 내용 분석 및 생성이 이루어집니다
- **학생 이름, 학번 등 개인 식별 정보는 전송되지 않습니다**
- 교육과정 성취기준, 학습 내용 등 교육적 텍스트만 처리됩니다
- API 응답 후 즉시 데이터가 삭제됩니다

---

## 3. 쿠키 및 로컬 스토리지

### 3.1 사용 목적

서비스는 다음 목적으로 브라우저 로컬 스토리지를 사용합니다:

- 작업 중인 교육계획 임시 저장 (브라우저 내에서만)
- 사용자 설정 저장 (테마, 표시 옵션 등)
- 모바일 관찰메모 임시 보관

### 3.2 관리 방법

- 사용자는 브라우저 설정에서 언제든지 로컬 스토리지 데이터를 삭제할 수 있습니다
- 서비스 내 "초기화" 기능을 통해 저장된 데이터를 즉시 삭제할 수 있습니다

---

## 4. 제3자 제공

IEPON은 **사용자의 개인정보를 제3자에게 제공하지 않습니다**.

단, 다음의 외부 서비스를 이용합니다:

### 4.1 OpenAI API
- **목적**: AI 기반 교육계획 생성 및 분석
- **전송 데이터**: 교육 내용, 성취기준 등 (개인 식별 정보 제외)
- **보관 기간**: 즉시 삭제

### 4.2 Vercel (호스팅)
- **목적**: 웹사이트 호스팅
- **수집 정보**: 일반적인 웹 서버 로그 (IP 주소, 접속 시간 등)
- **개인정보**: 수집하지 않음

사용자는 다음의 권리를 가집니다:

- 서비스 이용 중단 권리
- 개인정보 처리에 대한 문의 및 의견 제시 권리

## 6. 개인정보의 파기

IEPON은 서버에 개인정보를 저장하지 않으므로 별도의 파기 절차가 필요하지 않습니다.

사용자 브라우저에 저장된 데이터는:
- 사용자가 직접 삭제 가능
- 브라우저 캐시 및 쿠키 삭제 시 함께 삭제
- 서비스 내 "초기화" 기능으로 즉시 삭제 가능

## 9. 크롬 확장프로그램 (나이스 월별계획 입력 도우미)

IEPON 서비스와 연계된 크롬 확장프로그램은 다음 원칙을 따릅니다:

### 9.1 동작 방식
- 사용자의 **브라우저 내에서만** 작동합니다
- 외부로 데이터를 전송하지 않습니다
- 어떠한 정보도 저장하거나 수집하지 않습니다

### 9.2 권한 안내
확장프로그램이 요청하는 권한:

| 권한 | 목적 |
|------|------|
| `activeTab` | 현재 탭에서 작동하기 위해 필요 |
| `storage` | 사용자 설정(입력 속도 등) 기억 |
| `scripting` | 나이스 입력란에 내용 자동 입력 |
| `https://*.neis.go.kr/*` | 나이스 시스템에서만 작동 |

### 9.3 보안
- 모든 처리가 브라우저 내에서만 이루어집니다
- 학생 정보나 교육 내용이 외부로 전송되지 않습니다
- 최소한의 권한만 요청합니다
