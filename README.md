## 📌 서비스 소개

**IEPON**(아이피온)은 대한민국 특수교사를 위한 **개별화교육계획(IEP) AI 보조 웹 서비스**입니다.

2022 개정 교육과정을 기반으로 국어와 수학 교과의 **344개 성취기준**과 **교과서 단원 데이터**를 제공하며, AI 기술을 활용하여 특수교사의 IEP 작성 업무를 지원합니다.

### 핵심 가치
- 🆓 **완전 무료** - 모든 기능 무료 제공, 광고 없음
- 🔒 **개인정보 보호** - 학생 정보를 서버에 저장하지 않음
- 📱 **모바일 지원** - PWA로 앱처럼 사용 가능
- ⚡ **업무 효율화** - 나이스(NEIS) 원클릭 복사 지원

---

## 🎯 주요 기능

### 1. 현행수준 AI 분석
엑셀 파일을 업로드하면 AI가 학생의 현재 학습 수준을 자동 분석합니다.
- 국어·수학 영역별 학습 특성 파악
- 강점·약점 분석
- 맞춤형 교수전략 제안

### 2. 교육계획 자동 생성
AI 분석 결과를 기반으로 월별·학기별 교육계획을 자동 생성합니다.
- 2022 개정 교육과정 성취기준 기반
- 학생 수준에 맞는 목표 자동 선정
- 나이스(NEIS) 양식에 맞춘 출력

### 3. 교과서 단원 연계 ⭐
IEPON만의 핵심 기능으로, 성취기준과 함께 **교과서 단원 데이터**를 제공합니다.
- 월별 교과서 단원 배정 기능
- 교육과정-교과서-IEP 연계성 확보
- 실제 수업 진도에 맞춘 계획 수립

### 4. 나이스 원클릭 복사
생성된 모든 텍스트를 한 번의 클릭으로 복사하여 나이스(NEIS)에 바로 붙여넣을 수 있습니다.

### 5. 모바일 관찰메모 📱
모바일 전용 인터페이스(`/m`)에서 학생 관찰 메모를 실시간으로 기록합니다.
- 수업 중 간편한 메모 작성
- 학기말 평가 시 참고 자료로 활용
- PWA 설치로 앱처럼 빠른 접근

### 6. 교육평가 보조
학기말 교육평가 작성 시 AI가 성취도와 변화를 분석하여 평가 문구를 제안합니다.

## 📖 관련 용어

- **IEP (개별화교육계획)**: Individualized Education Program, 장애학생의 개별적 교육 요구에 맞춘 교육계획
- **특수교육**: 특별한 교육적 요구가 있는 학생을 위한 교육
- **나이스 (NEIS)**: 교육행정정보시스템, 대한민국 교육부 운영
- **2022 개정 교육과정**: 2022년에 개정된 대한민국 국가 교육과정
- **기본 교육과정**: 특수교육 대상 학생을 위한 교육과정
